## Trader Performance vs Market Sentiment Analysis
# Objective
Analyze how market sentiment (Fear/Greed) impacts trader behavior and performance.

# Methodology
*   Data cleaning and preprocessing
*   Merging sentiment and trade datasets
*   Feature engineering (PnL, win rate, trade frequency, etc.)
*   Behavioral and performance analysis
*   Segmentation of traders
# Key Insights
*   Win rates are highest during Extreme Greed and lowest during Extreme Fear
*   Low-frequency traders outperform high-frequency traders consistently
*   Traders adjust long/short positions based on sentiment
# Strategy Recommendations
*   Reduce overtrading to improve profitability
*   Increase participation during favorable sentiment (Greed) and reduce exposur  during Fear
*   Align trade direction with sentiment trends

*   Open the notebook in Google Colab
*   Upload datasets
*   Run all cells
